### 3. Скобки и Разделители
*Важно: чтобы скобки растягивались под высоту формулы (например, дроби), используйте `\left` и `\right`.*

| Название | Описание | Код | Результат |
| :--- | :--- | :--- | :---: |
| **Круглые скобки** | Обычные | `( a )` | $(a)$ |
| **Квадратные скобки** | Обычные | `[ a ]` | $[a]$ |
| **Фигурные скобки** | Требуют экранирования | `\{ a \}` | $\{ a \}$ |
| **Авто-размер** | Подстраиваются под контент | `\left( \frac{a}{b} \right)` | $\left( \frac{a}{b} \right)$ |
| **Угловые скобки** | Для усреднения или бра-кет | `\langle x \rangle` | $\langle x \rangle$ |

| **Норма** | Двойная черта | `\|x\|` | $\|x\|$ |

### Примеры сложных формул для документации

**1. Коэффициент теплопередачи:**
Код:
```latex
K = \frac{1}{ \frac{1}{\alpha_1} + \frac{\delta}{\lambda} + \frac{1}{\alpha_2} }
```
Результат:

$$ K = \frac{1}{ \frac{1}{\alpha_1} + \frac{\delta}{\lambda} + \frac{1}{\alpha_2} } $$

**2. Система уравнений:**
Код:
```latex
\begin{cases}
  \dot{x} = \sigma(y-x) \\
  \dot{y} = x(\rho-z)-y \\
  \dot{z} = xy - \beta z
\end{cases}
```
Результат:

$$
\begin{cases}
  \dot{x} = \sigma(y-x) \\
  \dot{y} = x(\rho-z)-y \\
  \dot{z} = xy - \beta z
\end{cases}
$$

**3. Единицы измерения (с текстом):**
Код:
```latex
C_p = 4.187 \quad \text{кДж}/(\text{кг} \cdot ^\circ\text{C})
```
Результат:

$$ C_p = 4.187 \quad \text{кДж}/(\text{кг} \cdot ^\circ\text{C}) $$
